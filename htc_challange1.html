Welcome to HTC: Hack The Crap - Console Attack Challenge

Challenge description

Welcome to the Console Attack Challenge hosted on our HTC (Hack The Crap) subdomain! In this challenge, you will put your skills to the test by exploiting a web application vulnerability that allows you to execute console commands and gain unauthorized access to hidden features. This challenge is designed to help you understand command injection attacks, a common vulnerability in web applications. Your goal is to use a console attack to access a restricted section of our simple web page. To succeed, you will need to identify the vulnerable input point and execute the appropriate commands to reveal the hidden content.
